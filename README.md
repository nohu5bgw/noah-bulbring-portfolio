# Noah Bulbring — Portfolio

Personal portfolio website for [noahbulbring.com](https://noahbulbring.com).

## Stack

- HTML, CSS, JavaScript — no frameworks
- Inter font via Google Fonts
- Deployed on Vercel

## Sections

- **Hero** — name, tagline, CTA links
- **About** — background, stats
- **Experience** — timeline of roles
- **Certifications** — IBM, Google, OpenAI, JPMorgan, Bank of America, PwC, LinkedIn, Harvard Online
- **Projects** — Quitslip, AI tools, e-commerce, finance dashboard
- **Contact** — email, LinkedIn, GitHub

## Deploy

1. Import repo into [Vercel](https://vercel.com)
2. No build step required — deploys as static HTML
3. Point `noahbulbring.com` to the Vercel domain in DNS
